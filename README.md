# 🌐 Nexa

**Nexa** é uma rede social experimental desenvolvida como um projeto entre amigos, com o objetivo de aprender, testar ideias e explorar conceitos modernos de interação digital.

A proposta combina elementos inspirados em plataformas populares, trazendo uma experiência própria, leve e focada em conexão entre pessoas.

---

## 🚀 Objetivo

O projeto Nexa foi criado com foco em:

- Praticar desenvolvimento (frontend, backend e lógica)
- Explorar conceitos de redes sociais modernas
- Testar ideias de interação entre usuários
- Trabalhar em equipe e versionamento com Git

---

## 🧠 Conceito

A Nexa mistura funcionalidades comuns de redes sociais, como:

- 📸 Compartilhamento de posts (imagens/texto)
- 💬 Interação entre usuários (curtidas, comentários)
- 👥 Perfis personalizados
- 📰 Feed dinâmico
- 🔗 Conexão entre pessoas (seguidores/amigos)

Tudo isso com uma identidade visual moderna e um foco em simplicidade.
